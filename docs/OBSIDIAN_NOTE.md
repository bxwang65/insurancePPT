## 关键技术点

### 数据提取
- `get_text("dict")` 位置解析 (不用 `find_tables()`, 有截断问题)
- 产品路由: 识别文档类型 → 只跑该产品的专用解析器
- **fitz始终覆盖**: `if ft.行数 > 0 → 覆盖` (2025-06-12修复)

### 年龄提取
- 独立脚本 `extract_age.py`
- 支持5种格式: "年齡:1" / "1歲" / "年龄:\\nVIP\\n1" / "男/1/标准" / "ANB:1"
